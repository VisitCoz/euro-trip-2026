# Euro Trip 2026

Live site (GitHub Pages): **https://visitcoz.github.io/euro-trip-2026/**

A family trip planner (23 Jun – 22 Jul 2026, Cancún ⇄ Europe). Static HTML, no build step.

## Pages
- `index.html` — landing page with a button per person
- `hub.html` — full interactive plan (EN/ES toggle, getaway explorer)
- `emma.html` — Emma's plan (English)
- `papas.html` — for the parents (Spanish)
- `stays.html` — where to stay (with booking links)

## How to update
Edit any `.html` file and push:
```
git add -A && git commit -m "update" && git push
```
…or edit the file directly on github.com (pencil icon → Commit changes). The live site updates in ~1 minute.
